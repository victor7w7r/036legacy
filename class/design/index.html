Diseño Web

Informática | Diseño Web | HTML/CSS/JS

Disponibilidad y descripción


¡Bien! El curso está disponible para individual y en grupo.
¡Ten en cuenta! Este curso tiene prerequisitos de aprendizaje
¡Ups! No ofrezco encargos ni trabajos para sitios web.

Es cierto, cualquiera puede realizar su sitio web, de manera didáctica e intuitiva con las diferentes herramientas y aplicaciones que da internet, que obviamente no está nada mal y es una oportunidad que todos deben aprovechar.

Pero puede que haya casos en que necesites agregar widgets de manera manual, control de formularios e inserción de código de API, o claro, puedes crear tu página web desde plantillas HTML, para esto necesitas saber entornos de vistas, programación y maquetado.

El uso de entornos de desarrollo pueden variar, se usará brackets o Adobe Dreamweaver para el entendimiento.
